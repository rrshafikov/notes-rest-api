# notes-rest-api

### **НИТУ МИСИС, Институт компьютерных наук**

**Учебный проект:** *Разработка REST API «Заметки + Пользователи»*  
**Выполнили:** Руслан Шафиков, Никита Белов, Георгий Горин  
**Группа:** БИВТ-24-1  

Этот репозиторий содержит учебный групповой проект, выполненный в рамках дисциплины **«Программирование»**.

**Цель работы:** разработать серверное приложение, реализующее REST API для управления пользователями и их заметками.  
Проект демонстрирует навыки построения API, командной разработки, документирования и применения современных инструментов веб-разработки.
